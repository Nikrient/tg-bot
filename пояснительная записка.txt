1. Введение
Телеграм-бот "Викторина" разработан для проведения увлекательных викторин с возможностью игры как в одиночном режиме, так и в режиме для двух игроков. Бот предлагает вопросы различной сложности из разных категорий знаний, включая математику, историю, науку, географию и культуру.
Основные цели проекта:
Создание интерактивного инструмента для проверки знаний.
Развитие логического мышления и эрудиции пользователей.
Предоставление возможности соревноваться с друзьями.
2. Функциональные возможности
2.1. Основные функции
Выбор количества игроков (1 или 2)
Пользователь может играть один или пригласить друга для совместной игры.
Игра с кубиком (для двух игроков)
Угадывание четности выпавшего числа.
Угадывание точного числа (от 1 до 6).
Определение победителя на основе результатов.
Выбор сложности вопросов
Легкая (100 очков за правильный ответ).
Средняя (500 очков).
Сложная (1000 очков).
Выбор категории вопросов
Математика
История
Наука
География
Культура
Система подсчета очков
Начисление полного количества очков за правильный ответ.
Штраф в размере половины стоимости вопроса за неправильный ответ.
Сохранение результатов игры
Логирование в базу данных SQLite.
2.2. Дополнительные возможности
Приглашение второго игрока по ссылке (для режима 2 игроков).
Гибкая система перезапуска игры (возможность начать новую игру или выйти в меню).
Обработка ошибок (уведомление пользователя о проблемах и предложение перезапустить бота).
3. Техническая реализация
3.1. Структура бота
Инициализация
Создание базы данных (scores.db).
Загрузка вопросов из словаря QUESTIONS.
Основные состояния диалога
(
    SELECT_PLAYERS,      # Выбор количества игроков
    WAIT_PLAYERS,       # Ожидание второго игрока
    DICE_CHOICE,        # Угадывание четности кубика
    NUMBER_GUESS,       # Угадывание числа
    SELECT_TIER,        # Выбор сложности
    SELECT_TOPIC,       # Выбор категории
    ASK_QUESTION,       # Задание вопроса
    GAME_OVER           # Завершение игры
)
Механика игры
Для одного игрока: сразу переход к выбору сложности.
Для двух игроков: сначала угадывание кубика, затем вопросы.
Хранение данных
Временные данные игры хранятся в словаре games.
Результаты сохраняются в SQLite (game_id, timestamp, players, scores).
4. Инструкция по использованию
Запуск бота
Пользователь отправляет команду /start.
Выбирает количество игроков (1 или 2).
Для одного игрока
Выбирает сложность и категорию.
Отвечает на вопросы, набирая очки.
Для двух игроков
Первый игрок создает игру и отправляет ссылку второму.
Оба игрока угадывают четность и число на кубике.
Победитель раунда выбирает категорию вопросов.
Игра продолжается до конца всех вопросов.
Завершение игры
Выводятся результаты.
Предлагается сыграть снова или выйти.
5. Возможные доработки
Добавление мультимедиа (изображения, аудио) в вопросы.
Расширение базы вопросов.
Режим турнирной таблицы (топ игроков).
Поддержка большего количества игроков (3-4 человека).
6. Заключение
Бот "Викторина: Интеллектуальный поединок" представляет собой удобный и увлекательный инструмент для проверки знаний и соревнований между пользователями. Гибкая система вопросов и настройки сложности делают его интересным для разных возрастных групп.
Проект может быть расширен за счет новых категорий, вопросов и игровых механик.
